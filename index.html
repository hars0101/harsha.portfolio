Key Projects

Spaceship Titanic Rescue Mission

A classification project using ensemble models to predict passenger fates during a spacetime anomaly. This project achieved an accuracy of 80.68% using VotingClassifier with tuned RandomForest and GradientBoosting models.

Explainable Heart Disease Prediction

An ensemble-quantum machine learning approach utilizing Quantum Support Vector Classifier (QSVC) and SHAP for feature contribution analysis. Compared QSVC with classical models to enhance prediction reliability and interpretability.

Drone Fish Feeding System

An AI-enabled drone system developed using Jetson Nano to automate fish feeding operations. This project highlights how smart technologies can be applied to agriculture and aquaculture for efficiency and precision.

Waste-to-Energy Ideathon

An ideathon project that addresses waste management challenges by converting waste materials into sustainable energy. Focused on environmental impact, energy conversion efficiency, and economic feasibility.

ATM Security System

A security enhancement project integrating RF technology and facial recognition to ensure secure ATM transactions. Combines hardware and software components for real-time authentication and fraud prevention.

Interactive Census Data Dashboard

A web application built to visualize and analyze India's census data. Utilized responsive design principles and APIs to provide users with an interactive and informative experience.
